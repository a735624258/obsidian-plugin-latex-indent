# Obsidian LaTeX 缩进

在 LaTeX 公式环境中按 Tab 自动插入 `\qquad` 缩进。

## 功能

- `$$...$$` 或 `$...$` 公式内按 Tab → 插入 `\qquad `（2em 空格）
- 尾部自动加空格避免 KaTeX 数字误读
- 纯编辑器级扩展，不污染渲染

## 使用

1. 在行间公式或行内公式中输入光标
2. 按 `Tab` 键 → 自动缩进

## 安装

下载 `main.js`、`manifest.json`、`styles.css`，放入 `.obsidian/plugins/latex-indent/`，重启 Obsidian 启用。
